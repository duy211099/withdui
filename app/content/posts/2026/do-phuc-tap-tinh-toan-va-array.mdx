# Array
- Static Array
  - allocated memory at compile time for popular PLs (C++, Java,...)
- Dynamic Array
  - Size is fixed
  - Located in **stack memory** space normally
    - Stack (fixed memory, not stack data structure) vs Heap (can request for more memory) - OS related knowledge

- Dynamic Array
  - allocated at run time for popular PLs (Python, Java,...)
  - Size is not fixed
  - Located in **heap memory space** normally
  - Popular methods:
    - pop
    - append

- Indexing
- Insertion/deletion at beginning
- Insertion at ending
- Deletion at ending
- Insetion in middle
- Deletion in middle
- Wasted space
