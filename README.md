# WithDui

WithDui is a Rails + Inertia.js app with a modern React frontend.

## Features

- Mood tracking
- Notes
- 80-year life calendar
- MDX-powered blog with admin UI

## Tech Stack

- Ruby 3.4.7, Rails 8.1
- PostgreSQL
- Redis (Action Cable + cache)
- Inertia.js + React 19 + TypeScript
- Vite + Tailwind CSS 4
